## Main User/Operator Flows

- Operator searches a reviewed skill index.
- Operator installs a reviewed skill bundle disabled/untrusted by default.
- Operator views reviewed-source plugin metadata and clones manually if desired.
- External client submits work through local API, ACP, MCP server, or AG-UI/A2UI.

## Data Flow

Marketplace metadata is fetched, validated, and stored with provenance. Skill
bundles use existing skill validation and trust state. Protocol clients
authenticate, rate-limit, submit work to Runtime or registered actions, and
receive redacted responses.

## Authority Invariants

Marketplace discovery grants no trust. Remote code-bearing plugin install is
not supported. External clients never have more authority than local workspace
users and cannot approve their own confirmations.
