## Main User/Operator Flows

- Operator asks Allbert to research a URL or topic.
- Allbert requests confirmation for a new navigation scope.
- Browser plugin navigates, extracts, screenshots, or reads bounded documents.
- Operator reviews extracted evidence and trace/audit metadata.

## Data Flow

Runtime frames a browser/research action, Resource Access records
`browser://session/<id>` and target URL scopes, Security Central confirms the
operation, the browser plugin performs bounded work, and results render in the
workspace with redacted traces.

## Authority Invariants

Browser plugin process ownership is not core authority. Each operation has a
separate grant. A navigate grant does not authorize fill/submit/download.
Browser content does not auto-promote to memory.
