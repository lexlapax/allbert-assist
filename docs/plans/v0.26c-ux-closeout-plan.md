# Allbert v0.26c Workspace UX Closeout

Semver target: `0.26.2`.

Request flow: `docs/plans/v0.26c-request-flow.md`

## Purpose

v0.26c is a small point-release plan for the workspace UX items explicitly
deferred from the `0.26.1` closeout. It keeps the v0.26a/v0.26b convention:
lettered v0.26 follow-ups mature the core workspace shell without changing
the platform contract that later app milestones prove.

This milestone does not move StockSage product work forward. It only closes
the remaining core-workspace polish that would otherwise distract from v0.27.

## Scope

- Add a real tile inspector modal for existing workspace tiles, including
  readable tile body, provenance, trace link, and copy affordances.
- Add a thread switcher dropdown from the AppBar thread chip, including recent
  threads and a new-thread action through existing runtime boundaries.
- Verify multi-tab workspace sync with manual screenshots or browser-based
  smoke notes for tile and ephemeral updates.

## Non-Goals

- No schema, migration, catalog, settings, signal, permission, or Security
  Central changes.
- No new StockSage card renderers.
- No progress streaming; v0.27 owns StockSage app progress streaming.
- No drag-drop tile reordering, canvas snapshots, plugin-contributed workspace
  regions, or external UI protocol exposure.

## Milestones

- M1: Tile inspector modal renders existing tile metadata and body content.
- M2: Thread switcher dropdown lists recent threads and creates a new thread.
- M3: Multi-tab sync verification records the expected operator behavior.
- M4: Closeout updates roadmap/CHANGELOG notes for `0.26.2`.

## Test Plan

- Focused LiveView/component tests for inspector open/close, keyboard escape,
  thread switching, and new-thread navigation.
- Browser/manual verification for multi-tab sync in the same thread.
- Docs-only gate when only planning text changes: `git diff --check`.
- If implementation lands in this point release, run the focused web tests and
  the normal warning gate before release closeout.

## Definition Of Done

v0.26c is done when the deferred core-workspace UX items from the `0.26.1`
roadmap closeout have either shipped or been explicitly reclassified, and the
workspace shell remains contract-compatible with v0.26.
