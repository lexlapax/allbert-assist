# Allbert v0.26c Workspace UX Closeout

Semver target: `0.26.2`.

Request flow: `docs/plans/v0.26c-request-flow.md`

## Purpose

v0.26c is a small point-release plan for the workspace UX items explicitly
deferred from the `0.26.1` closeout. It keeps the v0.26a/v0.26b convention:
lettered v0.26 follow-ups mature the core workspace shell without changing
the platform contract that later app milestones prove.

This milestone does not move StockSage product work forward. It only closes
the remaining core-workspace polish that would otherwise distract from v0.27.

## Scope

- Add a real tile inspector modal for existing workspace tiles, including
  readable tile body, provenance, trace link, and copy affordances.
- Add a thread switcher dropdown from the AppBar thread chip, including recent
  threads and a new-thread action through existing runtime boundaries.
- Verify multi-tab workspace sync with manual screenshots or browser-based
  smoke notes for tile and ephemeral updates.

## Module Shapes

Both features are net-new UI (no `TileInspector`/`ThreadSwitcher` exist today).
They are LiveView state + components in the existing web app; no new substrate.

### Tile inspector

- New LiveComponent `AllbertAssistWeb.Workspace.Components.TileInspector`
  (modal; reuses the `FocusTrap` hook + the M32 `.workspace-approval-overlay`
  scrim styling).
- `AgentLive` gains `open_tile_inspector_id` assign + handlers:
  - `handle_event("open_tile_inspector", %{"tile-id" => id}, socket)` → assign id.
  - `handle_event("close_tile_inspector", _, socket)` → clear id.
- The tile kebab (from M33) gains an **Inspect** menu item.
- Renders: tile kind + title, full body (read-only), provenance (emitter +
  emitted_at + scope), trace link (reusing `:trace_link`), and CopyToClipboard
  chips for tile id + body.

### Thread switcher

- The AppBar thread chip (currently a CopyToClipboard target from M34) becomes
  a **switcher toggle**; copy-thread-id moves into the dropdown as an item.
- `AgentLive` gains `thread_switcher_open?` assign +
  `handle_event("toggle_thread_switcher", _, socket)`.
- Switch routes use `push_navigate` to `/agent?thread_id=<id>`. The
  `new_thread` event calls the existing conversation boundary,
  `AllbertAssist.Conversations.resolve_thread(%{user_id: user_id, text:
  "Workspace session", new_thread: true})`, then navigates to the returned
  thread id; the switcher owns no private thread-creation logic.
- Recent threads come from `AllbertAssist.Conversations` (existing context); the
  switcher renders a bounded recent list.

### DOM Ids And Events

- Inspect trigger (kebab item): `#workspace-tile-inspect-<tile-id>`, event
  `open_tile_inspector`. Modal: `#workspace-tile-inspector` (role=dialog,
  `phx-hook="FocusTrap"`); close event `close_tile_inspector` (+ Escape).
- Thread chip toggle: `#workspace-thread-switcher-toggle`, event
  `toggle_thread_switcher`. Menu: `#workspace-thread-switcher-menu` (role=menu).
  Recent items: `#workspace-thread-item-<thread-id>` (navigate). New thread:
  `#workspace-thread-new`, event `new_thread` (resolve new thread, then navigate).
  Copy-id item:
  `#workspace-thread-copy-id` (CopyToClipboard).

## Non-Goals

- No schema, migration, catalog, settings, signal, permission, or Security
  Central changes.
- No new StockSage card renderers.
- No progress streaming; v0.27 owns StockSage app progress streaming.
- No drag-drop tile reordering, canvas snapshots, plugin-contributed workspace
  regions, or external UI protocol exposure.

## Milestones

### M1 — Tile inspector modal

Goal: an Inspect action on any canvas tile opens a focus-trapped modal showing
the full tile body + provenance + trace link + copy chips. Depends on: nothing
(builds on v0.26 tiles + M32 overlay + M33 kebab). Parallel-safe with: M2.

Tasks: add `TileInspector` LiveComponent; add `open_tile_inspector_id` assign +
open/close handlers to `AgentLive`; add the **Inspect** kebab item; reuse the
overlay scrim + FocusTrap.

Tests: `agent_live_test` — Inspect opens the modal with the tile body; Escape +
close button dismiss; focus returns to the kebab; copy chips present.

Acceptance: inspector opens/closes, traps focus, shows body + provenance + trace
+ copy, and returns focus on close.

### M2 — Thread switcher dropdown

Goal: the AppBar thread chip opens a dropdown listing recent threads + a
new-thread action + copy-thread-id. Depends on: nothing. Parallel-safe with: M1.

Tasks: convert the thread chip to a switcher toggle; add `thread_switcher_open?`
assign + handlers; render recent threads from `Conversations`; switch existing
threads via `push_navigate`; handle `new_thread` by calling
`Conversations.resolve_thread/1` with `new_thread: true` and navigating to the
returned id; move copy-id into the menu.

Tests: `agent_live_test` — toggle opens the menu; a recent item navigates to its
thread; new-thread creates + navigates; copy-id item present.

Acceptance: switcher opens, lists recent threads, switches + creates threads
through runtime boundaries, and preserves copy-id.

### M3 — Multi-tab sync verification

Goal: confirm tile + ephemeral updates fan out across tabs of the same
(user, thread). Depends on: nothing (verification of shipped v0.26 behavior).

Tasks: run two `/agent` tabs on one thread; emit a tile + an ephemeral; record
that both tabs converge; capture browser screenshots / notes.

Pass/fail bar: a tile/ephemeral created in tab A appears in tab B within ~1s
without a reload. Record the result in the request-flow Milestone Update Log.

Acceptance: documented verification (screenshots or browser-smoke notes) showing
cross-tab convergence; any defect found is filed, not silently passed.

### M4 — Closeout

Goal: ship `0.26.2`. Depends on: M1–M3.

Tasks: bump version metadata to `0.26.2` (umbrella + 2 apps +
`App.CoreApp.version/0`); CHANGELOG `v0.26c` section; roadmap row update; this
plan's status + the flow's Milestone/Release logs.

Acceptance: full web gate green; version `0.26.2`; docs current.

roadmap closeout have either shipped or been explicitly reclassified, and the
workspace shell remains contract-compatible with v0.26.
