## Main User/Operator Flows

- Operator enables MCP server mode in Settings Central (default off).
- Operator chooses transport (stdio for editor/desktop integrations, HTTP
  streamable for network clients).
- Operator explicitly enables specific tools and memory namespaces (empty by
  default).
- External MCP client (e.g., Claude Desktop) connects.
- External client lists exposed tools/resources.
- External client calls a tool; effectful tool returns
  `:confirmation_pending`; operator approves through their workspace; external
  client receives the result.

## Data Flow

Inbound MCP request → transport adapter → tool/resource exposure filter →
`Actions.Runner.run/3` → Security Central → confirmation (if needed) →
execute → redacted MCP response. External client never receives raw settings,
secrets, signals, trace bodies, or confirmation storage.

## Authority Invariants

- External MCP clients never receive more authority than local workspace
  users.
- External clients cannot approve their own confirmations.
- Tool exposure is opt-in per Settings Central; an empty exposure is a valid
  configuration.
- Memory resource exposure is opt-in per namespace.
- Settings, secrets, raw runtime signals, raw trace bodies, and confirmation
  storage are never exposable.
- MCP request payloads cannot escalate to unconfigured tools.
- HTTP transport tokens are encrypted at rest.

## Handles / CLI Surfaces

- `mix allbert.mcp_server status`
- `mix allbert.mcp_server tools list`
- `mix allbert.mcp_server resources list`
- Settings Central `mcp_server.*` keys (writable through registered settings
  actions).

## Smoke Checklist

- Configure at least one external MCP client (Claude Desktop or Cursor)
  against a disposable Allbert Home.
- Confirm default exposure is empty.
- Explicitly enable one read-only tool and one memory namespace.
- Confirm tool call succeeds end-to-end.
- Confirm confirmation-needing tool returns `:confirmation_pending` and
  operator approval completes the call.
- Confirm external client cannot self-approve a confirmation.
- Confirm settings, secrets, and trace bodies remain inaccessible.
