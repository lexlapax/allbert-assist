## Main User/Operator Flows

- Operator installs Allbert from the packaged path on a clean host.
- Operator starts `allbert serve` and sees runtime/web/channels health go green.
- Operator completes QuickStart onboarding and reaches a first useful chat.
- Operator inspects security/confirmation/event state through web, CLI, and TUI
  surfaces.
- Operator exercises one real policy-bounded integration path.
- Operator exports/imports or upgrades an Allbert Home and verifies behavior.
- Operator uninstalls service/binary without deleting user data unless requested.
- Maintainer runs final docs, warning, drift, secret-scan, and `release.v063`
  gates.

## Data Flow

No new runtime data flow is introduced. The RC records redacted evidence from the
existing packaged binary, daemon, web workspace, onboarding, settings/profile,
TUI, integration, export/import, and uninstall paths. Evidence must use
disposable homes or explicitly declared test homes, never a real operator home.

## Authority Invariants

No new capability or authority. The RC validates that every product path remains
behind the same action/settings/security spine and that packaged entry points,
onboarding profiles, and web affordances grant nothing by themselves.

## Handles / CLI Surfaces

- Packaged install/uninstall commands.
- `allbert serve`, `allbert onboard`, `allbert tui`, `allbert admin ...`.
- Web `/`, `/workspace`, `/jobs`, `/objectives`, onboarding, settings/model, and
  operator panels.
- Export/import or upgrade commands.
- Secret/evidence leak scan.
- `mix allbert.test release.v063` deterministic gate.

## Smoke Checklist

- Install binary on a clean host/home.
- Start `allbert serve`; verify runtime/web/channels health.
- Complete web QuickStart onboarding and terminal onboarding smoke.
- Use browser control to verify `/`, `/workspace`, `/jobs`, `/objectives`,
  onboarding, settings/model, and operator panels.
- Use grouped CLI/TUI admin reads without raw `mix`.
- Exercise one real policy-bounded integration path.
- Export/import or upgrade a Home; verify behavior and redaction.
- Uninstall service/binary; verify Home preservation unless explicit data removal
  is requested.
- Run leak scans, docs gate, warning gate, and `release.v063`.
