## Main User/Operator Flows

- Operator starts CLI voice mode.
- Operator uses workspace microphone input.
- Telegram voice note is transcribed as text input.
- Discord voice support reads from configured channel context.
- Allbert returns optional TTS output through configured provider.

## Data Flow

Audio capture becomes a bounded media resource, STT creates text input for the
normal runtime, responses optionally call a TTS action, and traces record
redacted media metadata rather than raw audio by default.

## Authority Invariants

Voice is modality, not authority. Audio input does not bypass channel or
runtime policy. TTS is a registered action. Audio retention is explicit and
default off.
