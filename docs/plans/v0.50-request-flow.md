## Main User/Operator Flows

- Operator exports a disposable Allbert Home.
- Operator dry-runs import on a second machine/home.
- Operator reviews migration, secret, schema, and rollback diagnostics.
- Operator runs the v1.0 acceptance matrix.
- Maintainer verifies final RC evidence before v1.0.

## Data Flow

Export/import reads settings, secrets metadata, memory, skills, threads,
traces, audits, jobs, objectives, confirmations, plugins, marketplace metadata,
protocol settings, self-improvement suggestion state, and caches through
documented facades, writes dry-run diagnostics, and records rollback guidance.

## Authority Invariants

No new capability is introduced. Export/import does not expose raw secrets.
Dry-run diagnostics are redacted. Security eval sweep must cover every
v0.39-v0.49 capability class.

## Smoke Checklist

- Export disposable home.
- Import dry-run into a second home.
- Confirm secrets are redacted and migration policy is clear.
- Run the cross-surface eval sweep.
- Confirm v0.45 self-improvement suggestions remain inert after export/import.
- Confirm docs and gates are clean.
