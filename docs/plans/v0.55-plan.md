## Purpose

Re-layout the existing `/workspace` web surface so **chat is the primary surface**,
modeled on the ChatGPT, Claude, and Hermes (https://hermes-agent.nousresearch.com/)
agent UIs. This is a **re-skin / re-layout of the existing Surface substrate**
(ADR 0023/0024 kept), not a rewrite:
- Chat becomes the primary surface (today it is left-pane of a 2-pane chat+canvas).
- Ephemeral surfaces become **popups/modals/popovers** rather than inline panels.
- The canvas is **demoted** to a launcher/secondary destination.
- UI labels cleaned up — **"Conversations"** replaces the "threads" label
  (UI strings only; the internal `Conversations.Thread` name is unchanged, and
  the distinct volatile `Session.Scratchpad` concept is untouched — chosen to
  avoid the thread/session terminology collision).

## Scope (to be detailed in Phase B / R3)
- Confirm/clarify the Hermes reference; map ChatGPT/Claude chat-UX patterns
  (message-level affordances, artifacts/popovers, streaming, switchers) onto the
  Surface DSL + the component catalog.
- Ephemeral→modal/popover; canvas demotion; "Conversations" relabel migration
  (UI strings only).
- ADR 0024 amendment for the chat-primary layout + popup ephemerals.
- Acceptance + eval rows + `release.v055` + a real browser manual-validation pass.

## Non-Goals (preliminary)
- No removal of the canvas/ephemeral substrate (re-skin, not rebuild).
- No internal `Thread`→`Session` rename; no DB/schema rename.
- No model-generated UI code (Surface catalog stays the boundary).
