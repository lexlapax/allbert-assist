# v1.0 Handoff — Acceptance Matrix

Status: current (authored at v0.66 M11). v0.66 is the integrated product release
candidate; v1.0 (roadmap item 67) is the stability release + tiered public-contract
freeze + non-developer product launch. This note is the handoff: the frozen set of
acceptance-matrix inputs v1.0 inherits from v0.66, and where each is proven.

v0.66 uses two-layer verification (`docs/plans/v0.66-plan.md` Locked Decision 1): a
deterministic `mix allbert.test release.v066` gate for contract/routing/boundary
invariants, plus operator-attested evidence (`docs/validation/v0.66/`) for the
install/browser/model/cross-platform claims the gate cannot run in a checkout.

## Contracts v1.0 freezes (roadmap item 67)

- Tier 1: Runtime, Actions/permissions, Plugin, App, Settings Central schema shape,
  Allbert Home layout, Channel adapter boundary, Resource Access URI/grants.
- Tier 2: SurfaceProvider, Surface DSL (additive-only carve-out), workspace
  canvas/ephemeral substrate (minus single-consumer components).

## Open items carried to v1.0

The `PENDING-OPERATOR` rows above (Linux/WSL2 installs, real cross-version upgrade,

## DIT Milestones Before Freeze

DIT here means direct integration test: evidence that cannot be proven by the
checkout-bound `release.v066` gate and must run against real hosts, real packaged
artifacts, or real configured providers before the v1.0 freeze.

- **DIT-1: Cross-host install coverage.** Run the WSL2 install/serve walkthrough and
  attach the redacted transcript to `docs/validation/v0.66/`; confirm macOS and Linux
  `0.66.0` evidence remain current.
- **DIT-2: No-docs first-run completion.** Complete the fresh non-developer path from
  install -> QuickStart -> local model setup/download -> first useful chat without
  reading developer docs.
- **DIT-3: Model-backed local knowledge.** Keep a memory candidate and verify a later
  model-backed chat uses only `:kept` memory; connect notes and verify grounded
  note-backed answer/write flows with confirmations.
- **DIT-4: Live advanced-surface regression.** Exercise browser research,
  browser-research delegate, configured remote-channel outbound/inbound, MCP/OpenAI/ACP,
  and Plan/Build approval smokes; record unconfigured classes as scoped out or blocking.
- **DIT-5: Home portability and teardown.** Run a real older-Home upgrade or import
  round-trip, then uninstall the service/binary and verify Allbert Home preservation.
- **DIT-6: Source/release-line parity.** Treat `v0.66.0` as the packaged artifact line and
  `origin/main` after v0.66 M12/M13 as the freeze-prep source/evidence line; if a public
  source tag is required before v1.0, use a docs/source point tag with `[skip-artifacts]`.
