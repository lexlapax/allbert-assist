Status: **v1.0.4 published but macOS acceptance failed; signed
`v1.0.5-rc.1` published but WSL2 acceptance failed; M8 remediation and
immutable `v1.0.5-rc.2` are required. Stable `v1.0.5` is blocked on the RC.2
Sections F-H rerun.** This is the executable companion to
[v1.0.4-plan.md](v1.0.4-plan.md). It exists because published v1.0.3 omitted
an explicit external browser-runtime contract: macOS `browser_doctor` failed
with `Cannot find module 'playwright/package.json'`. The v1.0.3 tag is
immutable. v1.0.4 published the corrected packaging boundary, but its macOS
BEAM-launched browser crashed because the driver applied Erlang port option
`:hide` on Darwin. v1.0.5 carries the Windows-only port-option correction and
must repeat the complete binary ceremony and platform evidence.

artifacts use `v1.0.5`. Do not loosen the installer's tag-identity rule.
