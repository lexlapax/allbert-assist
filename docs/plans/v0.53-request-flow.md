## Main User/Operator Flows

- Operator configures WhatsApp, Signal, or Matrix.
- External channel event maps to a local user/session identity.
- Allbert replies through the same channel adapter boundary as Telegram/email.
- Approval Handoff renders through the adapter's declared primitive subset.

## Data Flow

Channel events are deduped, mapped to local identity, normalized into runtime
input, and rendered back through provider-specific delivery rules. Each
adapter declares its approval-primitive subset per the v0.52 ADR 0016
amendment.

## Authority Invariants

- Phone number, device session, or Matrix id is not permission authority.
- Channel metadata never grants action permissions.
- All effectful work still routes through registered actions, Security Central,
  confirmations, traces, and audits.
- WhatsApp, Signal, and Matrix adapters cannot invent new permission classes
  or confirmation shapes.
