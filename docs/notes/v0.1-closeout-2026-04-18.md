# v0.1 Closeout Note

Date: 2026-04-18

M1-M6 completed the kernel feature milestones, but that did not automatically make Allbert a wrapped v0.1 from a user perspective.

The remaining gap is release usability rather than missing kernel primitives:

- The current target user is a technical source-based CLI user, not a packaged end-user app audience.
- First-run setup needs to be guided rather than implied by editing markdown and config files manually.
- Filesystem trust remains explicit opt-in. We prefer guided root selection over permissive defaults so file tools do not silently overreach.
- v0.1 is not considered shipped until the documented setup path is exercised successfully against both Anthropic and OpenRouter from a clean home directory.

This closeout pass therefore adds three final milestones:

- `M7`: guided onboarding and setup UX
- `M8`: user-facing README and onboarding docs
- `M9`: clean-home release smoke and shipped-state gate

The important framing choice is that these are still part of v0.1. M1-M6 proved the runtime shape; M7-M9 are what make the release understandable and dependable for a real technical user.
