# v0.11 release readiness

Date: 2026-04-25

v0.11 is ready to release from source. The release ships the TUI operator surface, kernel-owned telemetry, configurable status-line items, adaptive memory routing, explicit episode/fact recall tiers, the optional semantic retrieval seam, and the review-first personality digest `LearningJob` seam.

- `cargo fmt --check`
- `env -u RUSTC_WRAPPER cargo clippy --workspace --all-targets -- -D warnings`
- `env -u RUSTC_WRAPPER cargo test -q`
- `env -u RUSTC_WRAPPER cargo run -q -p allbert-cli -- --help`

Release documentation validation passed:

- `allbert-cli jobs template disable personality-digest`

- temp-profile daemon start plus `allbert-cli telemetry --json`

## Release posture

- Fresh profiles default to the TUI; upgraded profiles preserve classic REPL unless opted in.
- `SOUL.md` remains seeded and operator-owned. `PERSONALITY.md` remains optional, lower-authority, absent by default, and installed only after explicit acceptance.
- Staged memory and episode recall are never treated as approved durable memory.
- Semantic retrieval is disabled by default and ships with a fake deterministic provider only; real embedding providers remain additive follow-up work.
- Personality digest uses a provider-free deterministic renderer in v0.11 while preserving the hosted-provider consent and draft/install envelope for future provider-backed renderers.
- v0.12 can now start from the shipped telemetry, approval, provenance, rollback, and learning-job seams.
